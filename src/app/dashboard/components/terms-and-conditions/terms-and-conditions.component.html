Términos y Condiciones de Uso

Al acceder y utilizar el Sistema de Reconocimiento de Plagas y Enfermedades en Caña de Azúcar ("el Sistema"), usted acepta cumplir con estos Términos y Condiciones de Uso. Si no está de acuerdo con alguno de estos términos, debe abstenerse de usar el Sistema.

1. Descripción del Sistema

El Sistema permite a productores agrícolas, técnicos o instituciones:

  • Capturar y analizar imágenes de cultivos de caña de azúcar.
  • Detectar plagas y enfermedades mediante inteligencia artificial.
  • Acceder a asesoría técnica y recomendaciones de manejo.

El acceso puede realizarse mediante plataformas web, móviles o dispositivos integrados.

2. Creación de Cuenta y Seguridad

Para utilizar ciertas funciones del Sistema, el usuario deberá crear una cuenta proporcionando información precisa y actualizada. El usuario se compromete a:

  • Mantener la confidencialidad de sus credenciales de acceso.
  • No compartir su cuenta con terceros.
  • Informar inmediatamente sobre cualquier uso no autorizado.

3. Uso Aceptable del Sistema

El usuario se compromete a utilizar el Sistema únicamente con fines agrícolas, académicos o de mejora productiva. Está prohibido:

  • Usar el Sistema para actividades ilícitas.
  • Modificar, distribuir o reproducir el software sin autorización.
  • Cargar contenido ofensivo, falso o que infrinja derechos de terceros.

4. Propiedad Intelectual

Todos los derechos de propiedad intelectual del Sistema, su diseño, código fuente, modelos de IA, marcas y contenidos pertenecen al equipo desarrollador o a la institución responsable. El usuario no adquiere ningún derecho de propiedad sobre el Sistema, más allá del acceso y uso autorizado.

5. Responsabilidades

  • Del usuario: Usar el Sistema de manera correcta, segura y conforme a estos términos.
  • Del desarrollador: Garantizar que el Sistema funcione conforme a lo previsto, aunque no se asegura una disponibilidad continua sin errores.

Importante:

El diagnóstico generado por el Sistema es una herramienta de apoyo, pero no reemplaza la inspección técnica o profesional en campo.

6. Limitación de Responsabilidad

El equipo desarrollador no será responsable por:

  • Pérdidas económicas derivadas de interpretaciones erróneas de los datos del Sistema.
  • Daños causados por mal uso del sistema o decisiones tomadas únicamente con base en sus resultados.
  • Fallos temporales por mantenimiento o errores técnicos.

7. Actualizaciones y Modificaciones

El Sistema puede ser actualizado en cualquier momento para mejorar su funcionamiento, corregir errores o añadir nuevas funcionalidades. Estos Términos también podrán modificarse. Se notificará al usuario sobre cambios relevantes.

8. Vigencia y Terminación

Estos términos permanecerán vigentes mientras el usuario mantenga una cuenta activa. El usuario puede cancelar su cuenta en cualquier momento. El desarrollador se reserva el derecho de suspender el acceso por incumplimiento de los términos.

Este acuerdo se rige por las leyes mexicanas (o del país correspondiente según el lugar de desarrollo). En caso de controversia, las partes se someterán a los tribunales competentes del estado de Veracruz, México.
